# C Programming Project 2

This repository covers fundamental concepts around data structures, dynamic memory allocation, pointers, function pointers, recursion, and system simulation using C.

## Question 1: Dynamic Student Record Engine

A menu-driven program that manages student records using structures, dynamic memory allocation, pointers, and function pointers. The program allows adding, displaying, sorting, and analyzing student performance using a custom analysis function.

View [source code](./question1/strecord.c)

---

## Question 2: Adaptive Text Intelligence Tool

A text analysis program that processes a user-entered paragraph using dynamic memory allocation and pointer-based string traversal. It builds word statistics using a structure and performs multiple analyses such as word count, longest word, and frequency detection.

The program also includes a custom analysis function that calculates the average word length.

View [source code](./question2/txtanalysis.c)

---

## Question 3: Callback-Based Device Monitoring Simulator

A simulation program that models a device monitoring system using structures, unions, dynamic memory allocation, and function pointers (callbacks). It processes multiple device types such as temperature, pressure, and voltage sensors using different callback functions.

View [source code](./question3/callbacks.c)

---

## Key Concepts Demonstrated

- Dynamic memory allocation (`malloc`, `realloc`, `free`)
- Structures and arrays of structures
- Unions for memory-efficient data storage
- Pointer traversal and manipulation
- Function pointers and callback systems
- String processing and character handling
- Modular programming in C
- Basic simulation of real-world systems
