# Dynamic Student Record Engine

## Overview

This project is a menu-driven program that manages student academic records using dynamic memory allocation, structures, pointers, pointer traversal, and function pointers. The program starts with an initial dataset of students and allows users to perform different operations for managing and analyzing student records.

## Features

The program provides the following menu options:

1. **Add Student** – Add a new student record.
2. **Display Students** – Display all student records.
3. **Sort Students by Grade** – Sort students in ascending order based on their grades.
4. **Sort Students by Name** – Sort students alphabetically by name.
5. **Calculate Average Grade** – Calculate and display the average grade of all students.
6. **Find Top Student** – Display the student with the highest grade.
7. **Search Student by Name** – Search for a student using their full name.
8. **Analyze Grade Variance** – Calculate the variance of student grades.
9. **Exit** – Exit the program and free memory.
