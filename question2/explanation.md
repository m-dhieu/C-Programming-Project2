# Adaptive Text Intelligence Tool

## Overview

This is a menu-driven program that performs text analysis on a user's paragraph. It uses dynamic memory allocation, structures, pointers, pointer traversal, and function pointers to process and analyze text efficiently.

The program stores word statistics and performs different analyses such as counting words, finding the longest word, identifying the most frequent word, and performing a custom analysis.

The program also displays memory addresses to demonstrate good memory usage.

---

## Features

The program provides the following menu options:

1. **Count Words** – Counts and displays total number of words in paragraph.
2. **Longest Word** – Finds and displays longest word.
3. **Most Frequent Word** – Identifies most appearing word.
4. **Display Word Statistics** – Shows all words stored with their frequency counts.
5. **Average Word Length** – Calculates and displays average word length.
6. **Reset Paragraph** – Clears current data and allows user to enter new paragraph.
7. **Exit** – Exits program and frees memory.

---

## Input Format

The program accepts a paragraph from user/book/article. The text is processed automatically by splitting it into words using pointer traversal.
